# Contact Fetcher
A tiny clipboard utility that extracts email addresses and US phone numbers from a large text, built for quick scraping and data collection.
## Features
### Finds multiple US phone numbers contacts:
#### With country code
+1-123-555-6789
+1 123-555-6789
001 123-555-6789
#### Without country code
(123) 555-6789
123-555-6789
123.555.6789
#### Without Area code
555-6789
555.6789
+1-555-6789
and more...
### Finds email addresses
### Takes text from clipboard

## Requirements
pyperclip
install with "sudo apt install python3-pyperclip" or "pip install pyperclip"
